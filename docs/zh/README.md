# ZKsync Era：以太坊扩容的 ZK Rollup

[![Logo](eraLogo.png)](https://zksync.io/)

ZKsync Era 是一个二层 Rollup 解决方案，利用零知识证明来扩容以太坊，同时不牺牲安全性和去中心化。  
由于它与 EVM 兼容（支持 Solidity/Vyper），以太坊上 99% 的项目无需重构或重新审计代码即可迁移。  
此外，ZKsync Era 还使用基于 LLVM 的编译器，未来将允许开发者使用 C++、Rust 以及其他流行语言编写智能合约。
