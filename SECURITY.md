# Security Policy

## Reporting a Vulnerability

If you've discovered a potential vulnerability in ZKsync Era, please report it to us via our bug bounty program. 

## Bug Bounty Program

We have a bug bounty program in place to reward security researchers for discovering and reporting vulnerabilities in our system. 

## Audit and Bounty Claim Process

To claim a bounty for a security audit, please follow these steps:

1. Review the sequencer code and identify potential vulnerabilities.
2. Create an audit report detailing the vulnerabilities and provide recommendations for remediation.
3. Submit the report to our bug bounty program.
4. Claim the bounty by creating an issue on our GitHub repository with the following information:
   - A description of the vulnerabilities discovered
   - A link to the audit report
   - Your GitHub username for bounty payment
